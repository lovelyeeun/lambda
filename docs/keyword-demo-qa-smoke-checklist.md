# 키워드 데모 스모크 체크리스트

대상:
- 사무용 의자
- 청소기
- 모니터
- 토너
- A4용지
- 노트북
- 데스크
- 포스트잇
- 태블릿
- 정수기

## 1분 체크

각 키워드마다 아래 5가지만 본다.

1. 검색 후 `의도 분석 → 검색 진행 → 조건 카드` 순서가 보인다.
2. 조건 카드 제출 전에는 추천 결과 카드가 보이지 않는다.
3. 조건 카드 제출 후에는 추천 결과 카드가 보인다.
4. `상세 보기` 클릭 시 우측 패널이 열리고 옵션 드롭다운이 보인다.
5. 후보 상품까지 포함해 상세 패널 전환 시 이전 상품 정보가 남지 않는다.

## 외부 상품 추가 확인

외부 상품이 있는 키워드는 아래도 본다.

- 스크래핑 진행 또는 수집 완료 상태가 자연스럽게 연결된다.
- 가격 비교/외부 마켓 맥락이 끊기지 않는다.

## 회귀 체크

- `내 활동 → 채팅` 이동 후 우측 패널 잔상이 없다.
- 다른 키워드로 연속 검색해도 이전 패널 상태가 남지 않는다.
- 브라우저 콘솔에 `[demo-scenario-check]` 경고가 없다.
